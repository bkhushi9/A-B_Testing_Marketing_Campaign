# Calculate mean and standard deviation for conversions on each platform
facebook_mean = data["Facebook Ad Conversions"].mean()
facebook_std = data["Facebook Ad Conversions"].std()
adwords_mean = data["AdWords Ad Conversions"].mean()
adwords_std = data["AdWords Ad Conversions"].std()

# Define thresholds for high and low conversion days
facebook_high_threshold = facebook_mean + facebook_std
facebook_low_threshold = facebook_mean - facebook_std
adwords_high_threshold = adwords_mean + adwords_std
adwords_low_threshold = adwords_mean - adwords_std

# Create new columns categorizing days based on thresholds
def label_conversion_category(conversion, high_threshold, low_threshold):
    if conversion > high_threshold:
        return 'High-Conversion Day'
    elif conversion < low_threshold:
        return 'Low-Conversion Day'
    else:
        return 'Average Day'

data["Facebook Conversion Category"] = data["Facebook Ad Conversions"].apply(
    label_conversion_category, args=(facebook_high_threshold, facebook_low_threshold))
data["AdWords Conversion Category"] = data["AdWords Ad Conversions"].apply(
    label_conversion_category, args=(adwords_high_threshold, adwords_low_threshold))

Hypothesis Testing

To test for significant differences in conversion rates between Facebook and AdWords, we can use a two-sample t-test or a proportion test. Here’s the approach:

Steps:

Calculate Conversion Rates: For each platform, calculate the conversion rate as the total conversions divided by total clicks.

Formulate Hypotheses: Null Hypothesis (H₀): There is no difference in conversion rates between Facebook and AdWords.

Alternative Hypothesis (H₁): There is a significant difference in conversion rates between Facebook and AdWords.

Run the Statistical Test: Use a two-sample proportion z-test to determine if the observed difference is statistically significant.

In [65]:
from statsmodels.stats.proportion import proportions_ztest

# Calculate total conversions and clicks for each platform
facebook_total_conversions = data["Facebook Ad Conversions"].sum()
facebook_total_clicks = data["Facebook Ad Clicks"].sum()
adwords_total_conversions = data["AdWords Ad Conversions"].sum()
adwords_total_clicks = data["AdWords Ad Clicks"].sum()

# Conversion rates
facebook_conversion_rate = facebook_total_conversions / facebook_total_clicks
adwords_conversion_rate = adwords_total_conversions / adwords_total_clicks

# Run the two-sample proportion z-test
counts = [facebook_total_conversions, adwords_total_conversions]
nobs = [facebook_total_clicks, adwords_total_clicks]
z_stat, p_value = proportions_ztest(counts, nobs)

# Store and display results
hypothesis_test_results = {
    "Facebook Conversion Rate": facebook_conversion_rate,
    "AdWords Conversion Rate": adwords_conversion_rate,
    "Z-Statistic": z_stat,
    "p-value": p_value
}

With a very low p-value (close to 0), we can reject the null hypothesis, suggesting a statistically significant difference in conversion rates between Facebook and AdWords. Facebook appears to have a significantly higher conversion rate compared to AdWords.

Run a hypothesis test on click-through rates (CTR) to determine if there’s a statistically significant difference in the click performance between Facebook and AdWords.

Set up Hypotheses:

Null Hypothesis (H₀): There is no difference in click-through rates between Facebook and AdWords.

Alternative Hypothesis (H₁): There is a significant difference in click-through rates between Facebook and AdWords.

Perform a Two-Sample Proportion Z-Test to see if the difference in CTRs is statistically significant.
